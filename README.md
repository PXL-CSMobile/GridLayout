# GridLayout

- Verwijder de standaard ScrollView uit MainPage.xaml
- Voeg een Grid toe aan de MainPage.xaml
- Maak de nodige rijen en kolommen aan
- Maak onderstaande voorbeeld zo goed mogelijk na
- De gebruikte kleuren zijn:
	- Groen: #FF00FF00
	- Blauw: #FF0000FF
	- Rood: #FFFF0000
	- Geel: #FFFFFF00
	- Oranje: #FFFFA500
